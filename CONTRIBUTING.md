# Contributing to RustChain Arcade

Welcome to RustChain Arcade! This project combines retro gaming with blockchain mining. We're excited to have you contribute.

## Quick Start

```bash
git clone https://github.com/Scottcjn/rustchain-arcade.git
cd rustchain-arcade
sudo ./install.sh
```

## Ways to Contribute

### 🐛 Bug Reports
- Use GitHub Issues to report bugs
- Include steps to reproduce, expected vs actual behavior
- Attach logs from `~/.rustchain-arcade/logs/`

### 💡 Feature Requests
- Open an issue with `[Feature Request]` prefix
- Describe the use case and proposed solution
- Consider backward compatibility

### 🎮 Game Compatibility
- Test with different emulators (RetroArch, RetroPie, Batocera)
- Report controller compatibility issues
- Add new game systems to `GAME_SYSTEMS.md`

### 📝 Documentation
- Improve README.md
- Add setup guides for different hardware
- Translate documentation

### 🧪 Testing
- Test on various Raspberry Pi models
- Report hardware compatibility
- Verify achievement tracking

## Code Standards

- Python 3.8+ required
- Use type hints where possible
- Run `python3 -m py_compile` before committing
- Follow PEP 8 guidelines
